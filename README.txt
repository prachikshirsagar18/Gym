FitLife Gym Website - Simple Static Project
-------------------------------------------

Files:
- index.html
- style.css
- script.js
- images/ (placeholder images)

How to use:
1. Unzip the package.
2. Open index.html in your browser.
3. Edit content/images as needed.

Want further customizations? I can:
- Replace placeholder images with real photos
- Add contact form backend (PHP/Node)
- Add booking/payment integration
- Export as PDF brochure or PPT
